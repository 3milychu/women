I. Summary Statistics

In 79 global sample populations, female manager representation averages 31.3%
Male manager representation averages 68.7%

II. Correlation Matrix

It is the proportion of women that have attained post-secondary education that correlates with female manager representation.The dependent variable Female Manager representation correlates strongest with the education difference between females and males.

Correlation between female manager representation and female higher education attainment itself is 0.13
Correlation between female manager representation and education difference between males and females is 0.4

III. Regression Model

In this case, the bi-variate analysis is used since the multi-variate analysis finds no other significant independent variable

The only statistically significant relationship
is between Female Manager representation and Education difference.

Surprisingly, the relationship between Female representation and female education itself
is not statistically significant.

For each 1% increase in proportion of females attaining post-secondary education,
female manager representation increases .57 percentage points.

IV. Final Thoughts

It is not how educated some females are that changes the landscape.

It is the perception of how educated most females in a population that changes the landscape.

For females, being an individual outlier does less to procure security than
striving towards collective excellence.
